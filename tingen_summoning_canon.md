# 廷根游戏 · 召唤/降临 正典考据 (Summoning Canon)

> *Lord of the Mysteries* (《诡秘之主》) 邪教召唤/降临神祇的正典流程，面向「廷根邪教召唤邪神」游戏设计。
>
> **来源与可信度**：本文经联网核查（中英文 wiki + 原著章节，74 条带来源论点，对抗式核查）。每点标 **【硬正典·已核实】** 或 **【存疑/粉丝推测】**。括号内为来源（LotM Wiki=英文 Fandom；诡秘/萌娘/百度=中文；原著标章号）。
>
> **最重要纠偏**：**安提哥努斯笔记本身不是「召唤外神的法器」**——它是 1 级封印物/愚者途径，只「承载家族遗产 + 暗中腐蚀读者 + 血脉召唤流转」。真正招致高位/未知存在「注意」的，是**独立的仪式魔法体系**。设计上务必把「笔记（方法/线索）」与「仪式（召唤行为）」当成两套系统。

## ① 总览：神如何被「召唤/降临」

- **【硬正典】神不能直接入现实**，只能投射 **化身(avatar)/傀儡(marionette)**，或夺取肉身/物件作 **降临容器(descent vessel)**；本体长期封存于自身 **印记(seal)** 或星界神国。容器可让神「无需通知任何人、无需仪式即离开印记」。
- **【硬正典】降临两段式**：低位格化身（常序列 1）先落地或入梦 → 「延迟转化」达临界后 **本体替换化身** 完全降临。（范例：1358 年不可避免之轮在 Cordu 村经降临仪式送下天使 Termiboros 作化身。）→ **反制点：杀化身 = 伤本体**，给玩家「本体替换前打断」的窗口。
- **【硬正典】宇宙三层**：现实 / 精神世界 / 灰雾；灰雾遮蔽视线、保护低序列者不被高序列存在与外神直视。
- **【硬正典】「最初造物主」无形屏障 = 第五纪末日背景**：屏障削薄 → 外神可侵入 → 降临增多 = 末日级风险。
- **【硬正典】信仰/祈祷 = 能量，尊名 = 地址**（此比喻非原著术语）：**尊名(Honorific Name)** 是祈祷抵达神的指向地址，真神用 **三段式**（真名 + 代名/伪饰 + 权柄秘语，如克莱恩「不属于这个时代的愚者 / 灰雾之上的神秘主宰 / 掌控好运的黄黑之王」）。位格越高响应范围越大。
- **【硬正典】古神可用「一句式尊名」指向，但「只有求死的人才念一句」。**
- **【硬正典】完全降临罕见且灾难**：耗能巨大、通道易被高位者察觉拦截；容纳过量 **源质/唯一性(Uniqueness)** 会「立刻引发宇宙自毁」，故神多被迫部分显现。

## ② 前置条件：施法者 / 知识方法 / 容器

- **【硬正典】占卜家(Seer) 与窥秘人是仪式专精途径**，天生高灵性、适合仪式魔法。
- **【硬正典】低序列者只能向「正统神灵」求助，绝不能接触未知存在**（老尼尔：「只要尝试过一次，你就会不可避免地滑向深渊」）→ 邪教召唤外神 **往往需高序列祭司主持，并以献祭弥补灵性不足**。
- **【硬正典·补漏】施法者「灵性储备」是仪式的物理燃料**（可耗尽、靠冥想 Cogitation 补充）→ 低序列越级最直接的失败原因是**蓝条不够**，而非「招来错的存在」。
- **【硬正典·关键纠偏】安提哥努斯笔记 = 1 级封印物、愚者途径、黑夜女神教会持有**；内禀仅「承载家族遗产」；真实内容（仅血裔可见）= 夜之国概况 + 三份愚者途径魔药配方 + 指向家族宝藏（极可能陷阱）的线索；副作用 = **暗中腐蚀翻阅者 + 血脉召唤驱动流转**（是「被吸引/流转」，**不是主动召唤外神**）。
- **【硬正典·补漏】封印物分级 0–3 级，0 级最高**（对应天使/序列 1 级危险）→ 这才解释为何 **0 级「阿勒苏霍德之笔」(0-08)** 能幕后编排整场廷根悲剧，而 1 级笔记不能召唤。
- **【硬正典·容器】以强大非凡者或孕体作容器让神降临是真实手段**（极光会让真实造物主降临三法兼示两种燃料：**情绪绝望 + 非凡者容器**）。
- **【硬正典·纠偏】极光会崇拜真实造物主/亚当（被绞死者途径，序列 0 真神），非外神，且是外神信徒之敌**（走「用容器让造物主降临」路线，不属「屠杀献祭拜外神」）。

## ③ 需收集的材料与祭品

- **【硬正典】魔药配方 = 主要材料（带非凡特性，用量严格）+ 辅助材料（缓冲负面，用量宽松）**；可用一份纯「非凡特性」替代主材 → **杀死同途径非凡者、收割析出的特性** 是获取最稀缺单项的正典途径。
- **【硬正典】裸吞特性极危险**（高概率死/疯）→ 辅料 + 仪式 + 扮演法消化前任宿主的精神烙印。
- **【硬正典】祭品三档**：活的普通人（第三好）< 带非凡特性者（第二好）< 半神（最好）；献祭等级须匹配召唤目标位格。
- **【硬正典】屠城式集体献祭确有**：汇聚全城 **灵魂** 到祭坛统一献祭，多由邪魔途径 **女巫** 施放强力 **诅咒**。
- **物理来源四类（类别为硬正典）**：① 神话生物器官/部位 ② 血液 ③ 矿物水晶 + 魔法花卉 ④ 怨灵残余灵性 + 场所/封印取得物。
- **【存疑】⚠ 非占卜家途径的精确「克数/毫升配方表」**：原著仅 **占卜家途径** 有完整全序列配方；网传其余途径详表 **部分为粉丝外推**，引用须标注。

## ④ 仪式步骤：场所 → 天时 → 祭坛器物 → 咒文 → 献祭/召唤 → 收尾

- **【硬正典·逐字】仪式三段式**：① 引起对应存在兴趣的 **献祭** ② 描述该存在的 **咒文** ③ 以简单格式 + **符号** 表达所求。
- **【硬正典·补漏】两类风险不同的「召唤」要分开**：召唤灵界生物（契约信使，常见低风险） vs 招致高位/未知存在注意（高风险：被注视/污染/降临）。
- **场所**：干净、无人、富含灵性、避官方；银刀激水、绕台走出「灵性之墙」隔离内外，先冥想蓄精神。
- **天时**：【硬正典】许多仪式绑天象，月亮途径须 **满月** 并凑齐月相材料。**【存疑】⚠「血月专属时机」是误读**——canon 的是 **「血月献祭」这一仪式类型**，不是「血月才能做」。
- **祭坛器物**：银匕首、蜡烛（最重要，每支 2 种神域材料）、香精/草药；占卜类另需清水/粗盐/仿羊皮纸；**灵数** 参与符号与蜡烛布置（2=世界与诸神，3=神物接触）。
- **咒文**：【硬正典】最好用赫密斯语；**语言越古老/越接近自然 = 效果越强但越易暴露**（可做风险旋钮）；三段式尊名精确指向；**禁亵渎**。具名仪式关键词（逐字）：献祭仪式 `pray/notice/offer/kingdom/gates/open`；赐予仪式 `…give…strength…`；血月献祭 = 兽皮上写尊名 + 画符，最好用非凡者血。
- **献祭/召唤**：按等级当场杀祭品让灵性污血染坛；**灵舞** 是死亡所「偏爱」的方式（**【存疑】其与特定外神绑定、「死神复活」属外推/死神教会独立支线**）。
- **收尾**：滴精油 → 烧符纸 → 谢神 → 先「我」后「神」自右向左熄烛 → 撤墙；**收尾被打断 = 通道不闭合 = 反噬↑**。

## ⑤ 成功 vs 失败/反噬

- **成功条件**：三段式齐备 + 咒文精确 + 祭品等级匹配目标 + 场所纯净不受扰 + 收尾尊崇有序 + 施法者灵性足够供能。神可不经魔药直接 **赐福(Boon)**，但临时、且受赐者会越来越像该神（被同化/腐化）。
- **【硬正典】外神赐神格难且易被毁；仅三位较易：堕落母神 / 欲望之母树 / 不定者。**
- **四类失败**：① 描述不清 → 存在「按自己意思」乱来；② 执行错误 → **反噬**（未消化特性当场失控变怪）；③ 招来错的/危险存在被其注意 → 被附身/诅咒/腐化；④ 被官方非凡者发现。
- **【硬正典·补漏】占卜是低序列者最常见的「暴露通道」**：廷根三人自杀正由 **「黑色占卜」** 触发；镜子占卜对第三方有利但使用者毫无防护暴露于未知视线之下。
- **【硬正典】念尊名 = 定位风险 + 亵渎反噬**：尊名自带「祈祷与自动响应」，可被反向定位；直接念神真名通常是亵渎并招反噬。
- **【存疑】⚠「旧日（序列 0 以上）晋升无需仪式」** 仅见百度百科，是本资料中来源最弱的一条。

## ⑥ 邪教 Agent 召唤神祇的步骤（本文核心 → 给游戏 agent 用）

一个**邪教 NPC（施法者 agent）**要把神召唤/降临，需按顺序完成下列步骤。每步标出 canon 依据 + 引擎里的状态前提/效果，方便直接写成 agent 的目标链与 `Action` 的 `check_preconditions → apply_effects`。

| # | 步骤 | canon 依据 | 引擎前提/效果（建议） |
|---|---|---|---|
| 1 | **取得方法（魔典/笔记）** | 仪式魔法需经文本/师承传授；笔记 = 配方+尊名线索（非召唤按钮） | 获得 `grimoire` 道具；翻阅累积 `corruption`、触发「血脉召唤」事件 |
| 2 | **解读目标尊名** | 尊名是祈祷「地址」，真神用三段式；念全/念错有别 | 由笔记 + 占卜拼出 `honorific_name`（占卜抬升 `noticed` 值） |
| 3 | **收集材料与祭品** | 三段式需「献祭」；祭品三档须匹配目标位格；主/辅材料 | 集齐 `implements`(银匕首/蜡烛/香/兽皮) + `sacrifice`(活人<带特性者<半神) + `materials`（港口走私 / 墓地·停尸尸源） |
| 4 | **前往仪式场所** | 干净、无人、富含灵性、避官方非凡者 | agent 移动到 `Cathedral Crypt`（教堂地穴） |
| 5 | **等待时机窗口** | 许多仪式绑天象（满月）；通道随屏障/readiness 削薄而开 | `corruption/readiness ≥ threshold`（满月窗口开启） |
| 6 | **布置仪式** | 银刀激水绕台走出「灵性之墙」；灵数布蜡烛、画符；兽皮写尊名 | 设 `altar_ready=true`、`spirit_wall` 完整度（=被打断难度） |
| 7 | **执行三段式** | ① 献祭 ② 咒文(三段式尊名+关键词，选语言=风险旋钮) ③ 符号 | 消耗施法者 `spirituality`（蓝条）；蓝条耗尽=中断；亵渎/格式错=失败模式① |
| 8 | **降临结算** | 化身先落地(序列1前哨) → 延迟转化 → 本体替换化身 | 成功：spawn 化身 → 计时「延迟转化」→ 本体降临 = **邪教胜/末日** |
| 9 | **收尾（若未被打断）** | 滴精油→烧符纸→谢神→熄烛→撤墙 | 闭合通道；被打断=召唤失败 + 反噬 |

**一句话目标链（可直接当 cult agent 的 `Goal` 序列）**：
`取得笔记 → 解读尊名 → 集齐(材料+祭品+耗材) → 去地穴 → 等窗口 → 布阵 → 献祭+咒文+符号 → 化身降临→本体替换`。

## ⑦ 玩家如何打断（对抗点，全部来自 canon 反制逻辑）

| 打断点 | 效果 | canon 依据 |
|---|---|---|
| 抢走/烧毁笔记 | 失去配方与尊名，无法继续 | 笔记是方法/线索载体 |
| 夺走/破坏材料或祭品 | 断灵性供给与「献祭」，召唤中止 | 三段式必含献祭 |
| 杀死/制服施法者 | 无人主持 | 低序列需祭司主持 |
| 破坏祭坛 / 撕开灵性之墙 | 通道无法成形 | 灵性之墙隔离内外 |
| 打断咒文念诵 | 链接建立中断 | 咒文须按格式完成 |
| 在收尾阶段介入 | 通道不闭合 → 召唤失败 + 反噬施法方 | 收尾礼仪必须有序 |
| **趁「延迟转化」窗口杀化身** | **化身死 = 伤本体**，阻止完全降临 | 杀化身/傀儡=伤本体 |
| 用敌方尊名反向定位 | 侦查/追踪邪教祭司 | 尊名自带自动响应、可被定位 |

> **三条全局风险条**（建议）：`被注意(noticed)` / `腐蚀(corruption)` / `官方追查(heat)`——任一爆表触发对应阵营介入（未知存在 / 教会·值夜者 / 治安），逼玩家在「快出力量」与「别被发现」间权衡。这正是廷根早期主线的可玩化身（0-08 之笔编排 → 笔记流转 → 雷·比伯吸收失控 → 值夜者击杀）。

## ⑧ 本轮联网核查修订（摘要）

- **升级为硬正典**：仪式三段式、祭品三档、蜡烛点/熄顺序、献祭/赐予仪式关键词、血月献祭做法、女巫屠城献祭、降临容器/两段式(Termiboros)、尊名作「地址」、笔记全部属性（及「笔记≠召唤法器」纠偏）、接触外神知识被注视/污染/腐化、三位易赐神格的外神、灵舞为死亡所偏爱。
- **纠正的混淆**：极光会≠拜外神（拜真实造物主，是外神之敌）；血族(月亮途径)≠受难会(拜必然之环)；「血月时机」≠「血月献祭(仪式)」；灵舞-特定外神绑定/死神复活为外推。
- **补上的遗漏**：封印物分级体系；占卜暴露子类；「召唤信使」vs「招致注意」两类风险；咒文语言层级=强度↔隐蔽权衡；灵性储备作物理燃料门槛。
- **保留存疑（引用须标注）**：非占卜家精确配方表、血月专属时机、灵舞-外神绑定、旧日晋升无需仪式、按外神归属的「邪教名册」（多为作者微信问答补充设定，非正文 canon）。
